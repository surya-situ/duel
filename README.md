## Install

> To install packages in both the server and frontend folder:
- cd server
```sh
npm install
```

- cd frontend
```sh
npm install
```

## Usage

- To run the server 
```sh
npm run dev
```
- For frontend 
```sh
npm run dev
```

## Run tests

```sh
npm run test
```
